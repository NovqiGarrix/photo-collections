## Photo Collections using Next.js

Using Pexels API to show images with infinite scroll component

## Techs
- Next.js
- TypeScript
- Tailwind CSS
- react-infinite-scroll-component

## Getting Started
To get started, see `src/pages/index.tsx`.

## Run in Local
```bash

# pnpm
pnpm dev

# yarn
yarn dev

# npm
npm run dev

```
